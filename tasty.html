Regulation

TastyFX is fully regulated by the CFTC and is a member of the NFA for US-based traders, which brings oversight and security requirements.

All US regulated forex brokers are subject to non-segregated accounts, FIFO (first in first out) and non-hedging. TastyFX handles the FIFO rule implementation by allowing trades to be opened of the same lot size and symbol; however, FIFO is enforced on closing trades. If multiple orders are opened for a given symbol and a stoploss is applied to one of the trades, the same stoploss level will be applied to the other open trade as well to avoid a FIFO conflict. This allows greater flexibility when opening a trade, and less flexibility when closing trades.
